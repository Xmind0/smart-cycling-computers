
# 智能码表软件

本项目是一个基于 Qt QML 开发的智能码表软件，旨在为用户打造一个集成多种功能的智能骑行伴侣。该系统设计用于 800x680 分辨率的设备，提供清晰直观的操作体验。
由于设备续航问题暂时没有想出很好的方法，目前考虑先做出软件部分。
## 核心功能

* **基本信息显示:**
    * 实时时间显示，支持自定义样式。
    * 集成天气服务，实时更新温度、湿度、风力等信息。
    * 显示日期、电池电量、信号强度等设备信息。


* **网易云音乐播放:**
    * 集成网易云音乐 API，在线搜索和播放音乐。
    * 支持本地音乐播放，兼容常见音频格式。
    * 提供播放控制功能：播放/暂停、上一首/下一首、音量调节。
